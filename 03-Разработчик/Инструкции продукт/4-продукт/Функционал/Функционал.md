Несколько **мобильных приложений** (и при необходимости **веб-клиентов**) для разных ролей на одной **платформе**. Термины: [Глоссарий](../../../../01-Директор/Инструкции/1-контекст-и-правила/Глоссарий.md).

**Первая коммерческая рамка (модули MVP, тарифы Light / Pro / Max, партнёрка, каналы продаж):** [MVP — модули и тарифы](MVP%20—%20модули%20и%20тарифы.md). Ниже — целевое видение «полного фарша» без привязки к конкретному тарифу.

## Для клиента
- план тренировок,питания
- готовые рецепты с КБЖУ
- трекер питания, тренеровок, сна, воды, самочувствия
- напоминание о встрече
- напоминание о трекере
- общение со специалистом
- видео тренеровок
- загрузка фото еды
- возможность оплатить услуги специалиста
- Отображение динамики
- мотиваторы достижений
- Обратная связь (что улучшить в продукте, что неудобно, ошибки)

## Для специалиста
- Электронный журнал занятий (календарь)
- ЦРМ
- Чат с клиентами (с возможностью привязки к упражениеям или планам)
- Заметки по клиентам
- Автоматическое оповещения клиентов о встрече со специалистом
- Напоминания по трекерам
- Планы тренировок и питания
- готовые рецепты с КБЖУ
- готовые шаблоны питания, тренеровок
- Конструктор шаблонов питания
- Конструктор шаблонов тренировок
- Загрузка видео тренеровок (возможно, ссылок на видеохостинги)
- Трекер тренеровок, питания, воды, активности, сна, замеров. С показателями отклонений от запланированного.
- Прием оплаты через платформу. Запись в Мой налог. Потенциально удержание и оплата налога.
- Рекомендации по удержанию клиентов; идеи по механикам — см. [Идеи-удержание-клиента-тренера](Идеи-удержание-клиента-тренера.md)
- Инструменты для привлечения клиентов
- Расчет бизнес-показателей
- акции
- Автоматический рассчет КБЖУ, тоннажа, риска ССЗ, других показателей
- Прогресс клиента визуально (графики)
- Персонализация: загрузка фото, установка цветов интерфейса
- Функционал для работы с группой (оплаты, оповещения, чат)
- Статьи. Текстово-графическая микроинформация, чтобы поделиться с клиентом.
- Обратная связь (что улучшить в продукте, что неудобно, ошибки)

## Для маркетолога (после MVP)
- ЦРМ
- Финансовые показатели
- Обратная связь (что улучшить в продукте, что неудобно, ошибки)

## Для поддержки (после 500-1000 платящих)
- ЦРМ
- Показатели использования специалистом инструментов сервиса
- Чат со специалистом
- Обратная связь (что улучшить в продукте, что неудобно, ошибки)


## Для руководителя
- Показатели работы компании



NB: на этапе MVP отдельные сборки для маркетолога и поддержки не делаем; при росте можно объединить их в одну внутреннюю сборку.

- Пуш-сервис
- Сервисы по расчету коллаража
