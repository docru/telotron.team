# Примерная рхитектура

## Сервер
один VPS (рекомендуется 4–8 ГБ RAM) с Docker и Docker Compose.

## Компоненты
### Платформа в Docker (4 контейнера):

Frontend

Backend

Redis

MySQL

### Стек логирования (Loki + Grafana):

Promtail — агент, собирает логи из всех контейнеров (через Docker socket)

Loki — хранилище логов, легковесное, индексирует только метки

Grafana — визуализация и алерты, подключена к Loki как datasource

### CI/CD:

GitLab (Self-hosted) — репозиторий и CI/CD сервер

GitLab Runner — исполняет пайплайны, собирает образы и деплоит
