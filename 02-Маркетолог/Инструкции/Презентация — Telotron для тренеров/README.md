## Сборка PDF

Из каталога `Инструкции/`:

```bash
python3 build-presentation-pdf.py
```

Собирает **v1.3** → `Презентация — Telotron для тренеров/Презентация — Telotron для тренеров v1.3.pdf` (и `.docx`).

Архив v1.2:

```bash
python3 build-presentation-pdf.py --version 1.2
```
