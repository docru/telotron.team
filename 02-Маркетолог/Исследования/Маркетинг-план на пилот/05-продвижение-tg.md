# Продвижение Telegram · операционный план

**Статус:** **опционально** (не в фокусе директора на Пилоте) · черновик v0.1 · **2026-06-12**  
**Канон приоритетов:** [Продвижение — фокус пилот 2026](../../../01-Директор/Инструкции/Продвижение%20—%20фокус%20пилот%202026.md) — TG **после** личных связей и VK.
**Старт:** **15.06.2026** · вклад TG в цель пилота: **~15–25 reg** из **~70** соцсетей к 31.07  
**Родитель:** [03 · соцсети (общее)](03-продвижение-соцсети.md) · **стратегия:** [ручное продвижение 2026–2027](../Стратегия%20роста%20—%20ручное%20продвижение%202026-2027.md) — **без paid** до ~50–100 платников.

**Термин:** **продвигаем** — CTA → `telotron.ru` → партнёрская ссылка **`/i/{token}`**.

---

## 1. Роль Telegram в пилоте

| TG — не для | TG — для |
|-------------|----------|
| Массовых рассылок в чужие чаты | **B2B:** тренеры, которые уже живут в мессенджерах |
| Дублирования VK один в один | **Короткий дайджест** + **личка** тёплым |
| Замены канала MAX (auth) | **Чаты тренеров** — ответы по делу, не спам |
| Контента «для клиента зала» | **Доверие основателя** + переход + **reg** |

**Telegram — второй соцканал** пилота (**1 пост/нед** в канале); основной объём reg из соцсетей — [VK](04-продвижение-vk.md). В TG сильнее **личка** и **участие в чатах**, чем «виральность канала».

**MAX в этом документе не рассматриваем** — только **auth/OTP** в продукте, не канал постинга ([03 · §2](03-продвижение-соцсети.md)).

## 2. Архитектура: личный аккаунт + канал + чаты

**Решение:** **гибрид**, как во VK — не отдельный «маркетинговый» аккаунт с нуля.

| Объект | Назначение | Имя / формат |
|--------|------------|--------------|
| **Личный аккаунт** | Лицо основателя, личка тёплым, пересылки знакомым | **Алексей Русаков** · `@username` *(если есть)* |
| **Публичный канал** | Дайджест: пилот, скрины, FAQ, одна ссылка reg | **«Telotron · для тренеров»** · `t.me/telotron_trainers` *(или свободный @)* |
| **Чаты тренеров** | Присутствие без рекламы: **ответ по теме** → «могу в личку» | поиск + рекомендации ядра |

**Почему личный аккаунт — да**

- Тренеры привыкли писать **человеку**, не боту.
- Пересечение с **личными связями** и церковными сетями — тот же основатель, что во [VK](04-продвижение-vk.md).
- На пилоте **15 минут провести по reg** — удобнее из лички TG.

**Почему канал — обязательно**

- Одна **закреплённая** точка: что такое Telotron, пилот, ссылка.
- Не смешивать продуктовые посты с личной перепиской.
- Партнёры-ядро (с ~01.07) могут **пересылать пост канала** со своим `/i/{token}` в подписи *(осторожно, не спам)*.

**Чего не делаем**

- **Отдельный маркетинг-бот** для рассылок — путаница с **продуктовым** Telegram-ботом (OTP, уведомления).
- Массовый **forward** поста канала в 10 чатов подряд — бан и репутационный минус.

**Схема**

```
Личный TG                 Канал «Telotron · для тренеров»     Чаты тренеров
──────────                ───────────────────────────────     ───────────────
Личка 1:1                 1 пост/нед (короткий)               Коммент по теме
«проведу за 15 мин»       Закреп: пилот + ссылка              Без ссылки в лоб
Пересылка канала знакомым Скрины альбомом / медиагруппой      «есть инструмент — в личку»
```

## 3. Отличия TG от VK (кратко)

| | VK | Telegram |
|---|-----|----------|
| **Ритм** | 2 поста/нед | **1 пост/нед** в канале |
| **Длина** | 800–1500 знаков | **300–700** знаков + ссылка |
| **Охват** | лента + группы | подписчики канала + **личка + чаты** |
| **Главный рычаг** | комментарии в группах VK | **чаты тренеров** + личка |
| **Медиа** | альбомы в группе | **медиагруппа** 2–4 скрина |

**Правило контента:** один смысл — **адаптация**, не копипаст. VK-пост из [04 · §9](04-продвижение-vk.md) → сжать для TG (§9).

## 4. Чеклист до 15.06

### 4.1 Личный аккаунт (~15 мин)

- [ ] **Bio** — блок про Telotron (текст §9.1); личное не удалять.
- [ ] **Аватар** — узнаваемое фото (можно то же, что VK).
- [ ] **Ссылка** в профиле → канал или `telotron.ru`.
- [ ] **Партнёрский токен** `/i/{token}` для учёта reg ([Ц2-17](01-цели-этапа-pilot.md)).

### 4.2 Канал «Telotron · для тренеров» (~30 мин)

- [ ] Создать **публичный канал**; короткий `@username` (латиница).
- [ ] **Описание канала** — текст §9.2.
- [ ] **Аватар** — логотип Telotron.
- [ ] **Закреп:** приглашение на пилот + ссылка reg + «без оплат до 01.08» (§9.3, укороченный).
- [ ] **Первый пост** — запуск (§9.4).
- [ ] Упомянуть канал в **VK** «О себе» и первом посте ([04 · §4](04-продвижение-vk.md)).

### 4.3 Чаты и операционка (~ongoing)

- [ ] Список **5–10 чатов** тренеров/нутрициологов (название, правила, активность) — таблица §8.1.
- [ ] Журнал: дата · канал/чат/личка · тема (A/B/C/D) · reg по токену.
- [ ] До Filament (**Ц2-17**) — отдельный токен или метка «источник: TG-канал / TG-личка / TG-чат».

## 5. Воронка и атрибуция

```
Пост в канале / личка / ответ в чате
  → ссылка /i/{token}
  → telotron.ru
  → reg Pro
  → activation (цель 1)
```

**В канале:** одна ссылка в посте; дублировать «напишите мне в личку — проведу за 15 минут» (@username).

**В чатах:** **без** ссылки в первом сообщении, если правила чата запрещают рекламу — только «могу показать в личку».

**Не обещать:** цены, списания, доход от партнёрки — только **триал 60 дней**, **полный функционал**, **без оплат до 01.08**, **нужна честная ОС**.

**Про Telegram в продукте:** в FAQ можно сказать, что **OTP и напоминания** идут через Telegram (или MAX / e-mail) — это **функция продукта**, не «подпишитесь на наш канал».

## 7. Работа в чатах тренеров

### 7.1 Правила

1. **Сначала польза** — ответ по вопросу (учёт клиентов, таблицы, напоминания).
2. **Потом мягко** — «мы как раз это автоматизируем в пилоте, могу в личку показать».
3. **Не** постить ссылку telotron.ru в чат, если не спросили и если правила против рекламы.
4. **Не** писать одно и то же в 5 чатах в один день.
5. **Не** добавляться в чаты только ради рекламы — 1–2 осмысленных сообщения в неделю на чат достаточно.

### 7.2 Где искать чаты (черновик)

| Тип | Примеры поиска | Действие |
|-----|----------------|----------|
| Тренеры / фитнес | «персональный тренер чат», «тренеры [город]» | вступить, **прочитать правила** |
| Нутрициологи | смежные чаты (пересечение ЦА) | только если есть экспертиза в теме |
| Локальные | Краснодар, Ейск, край | осторожно, без массового копипаст |
| Рекомендации | у тренеров из **личных связей** | приоритет — «живые» чаты |

**Таблица §8.1** — заполнить по мере нахождения; PO + директор.

### 7.3 Два сценария

| Сценарий | Кто | Как |
|----------|-----|-----|
| **Тёплый** | Знакомые, церковные сети | **Личка** + пересылка поста канала; без чатов |
| **Холодный** | Чужой чат тренеров | Ответ по теме → личка → reg; канал — «подробности здесь» |
