# Продвижение в соцсетях · операционный план (часть 3)

**Статус:** черновик v0.1 · **2026-06-12**  
**Старт:** **15.06.2026** · цель пилота: **~70 reg** из соцсетей к 31.07  
**Стратегия роста:** [ручное продвижение 2026–2027](../Стратегия%20роста%20—%20ручное%20продвижение%202026-2027.md) — **без paid** до ~50–100 платников.

**Термин:** **продвигаем** (не «продаём в личке») — CTA → `telotron.ru` → **партнёрская ссылка** `/i/{token}`.

---

## 1. Зачем соцсети в вашей модели

| Не для этого | Для этого |
|--------------|-----------|
| Массовый охват «всех фитнесов» | **Найти тренеров**, похожих на ЦА пилота |
| Красивый бренд ради лайков | **Доверие** + **переход** + **reg** |
| Конкуренция с блогами про тренировки | **B2B:** «день тренера без хаоса в чатах» |

**Соцсети — усилитель** личных связей: кейсы и посты **поддерживают** личку, не заменяют её до **~01.07**.

**С ~01.07:** **1–3 помощника** — копирайт/публикация/комментарии по **шаблонам**; стратегию и спорные формулировки — **директор**.

---

## 3. Воронка

```
Пост / сторис / комментарий
  → ссылка с меткой (партнёрский /i/{token} или платформенный)
  → telotron.ru (A13)
  → reg Pro
  → activation (цель 1)
```

**Меряем:** переходы → **reg по токену** (Filament — доработка **Ц2-17**); до dev — UTM + таблица.

**Не обещаем в постах:** цены, выплаты партнёрки, «заработок» — только **триал 60 дней**, **без оплат до августа**, **ищем обратную связь**.

---

## 4. Контент: 4 опоры (чередовать)

| # | Опора | Пример темы |
|---|--------|-------------|
| **A** | **Боль** | Таблицы, 5 чатов, «где план клиента?» |
| **B** | **Как устроено** | 3–5 скринов: календарь, клиент, план |
| **C** | **Пилот / приглашение** | «Ищу 10–15 тренеров на 60 дней, нужна ОС» |
| **D** | **FAQ / снятие страха** | Passkey, MAX, триал, **без оплат до 01.08** |

**Соотношение в пилоте:** **A+C** чаще, **B** когда есть скрины, **D** раз в 2 недели.

**Тон:** от **первого лица** основателя («мы запускаем», «ищу коллег») — не безликий SaaS.

## 6. Шаблон поста (VK/TG)

**Структура:**

1. **Крюк** (1 строка боли): «Клиент спросил план — вы ищете в переписке месячной давности?»
2. **Что делаем** (2–3 буллета): запись · планы · дневник клиента.
3. **Пилот:** 60 дней, полный функционал, **без оплат до августа**, нужна честная ОС.
4. **CTA:** ссылка → регистрация; «напишите в личку — проведу за 15 минут».

**Длина VK:** 800–1500 знаков. **TG:** короче + ссылка.

---

## 7. Где брать аудиторию (без бюджета)

1. **Личная сеть** — репосты знакомых тренеров (пересечение с частью 2 плана).
2. **VK:** поиск сообществ «фитнес тренер», «персональный тренер», городские — **не** спам в стену, **коммент** по теме.
3. **TG:** чаты тренеров / нутрициологов — **правило:** сначала польза, потом «есть инструмент, в личку».
4. **Церковные / локальные сети** — через **личное** представление, не холодный пост.

## 8. KPI (пилот)

| Метрика | Ориентир к 31.07 |
|---------|------------------|
| Reg из соцсетей | **~70** (цель 2) |
| Постов опубликовано | **≥12** VK + **≥6** TG |
| Переходы на сайт | рост неделя к неделе (abs. число TBD) |
| **Activation** из social-reg | **≥40%** (как общая цель 1) |

**Не KPI:** лайки, подписчики канала (вторично).

---

## 9. Чего не делать

- Массовые рассылки «в лоб» в чужие чаты.
- Обещать **доход** от партнёрки до **01.08**.
- Контент «для клиента конечного» (только **B2B** в пилоте — [цель 5](01-цели-этапа-pilot.md)).
- Ждать «идеального видео» — **текст + скрин** достаточно до 01.08.
