## 1. Роль VK в пилоте

| VK — не для | VK — для |
|-------------|----------|
| Массового «вирального» охвата | **B2B:** найти тренеров, похожих на ЦА |
| Конкуренции с фитнес-блогами | **Доверие основателя** + переход + **reg** |
| Замены личных касаний в июне | **Усилитель** лички: посты и кейсы подкрепляют 1:1 |

**VK — основной соцканал** пилота (2 поста/нед); Telegram — дайджест (1 пост/нед). Подробнее: [05 · TG](05-продвижение-tg.md).

## 2. Архитектура: личный профиль + сообщество

**Решение:** **гибрид**, не отдельный «пустой» личный аккаунт.

| Объект | Назначение | URL / имя |
|--------|------------|-----------|
| **Личная страница** | Лицо основателя, доверие, репосты знакомых, личка | [vk.com/id224642120](https://vk.com/id224642120) · **Алексей Русаков** |
| **Сообщество** | Контент продукта: скрины, FAQ, закрепы, ссылка на reg | **«Telotron · для тренеров»** *(создать до 15.06)* |

**Почему личный профиль — да**

- Живая личность (не безликий бренд) — важно для пилота и «ищу коллег на обратную связь».
- Пересечение с **личными связями** и **церковными сетями** (Ейск / Краснодар) — канал из [плана продвижения §2](../../План%20продвижения%20—%20этап%201%20Пилот.md).
- **IT-бэкграунд** (КубГТУ, 2000) — легитимность «делаю сервис для тренеров».
- **101 подписчик** — достаточно для ручного старта; рост не KPI.

**Почему сообщество — обязательно**

- В личной ленте много **семейного и духовного** контента — холодному тренеру нужна **точка входа «про продукт»**.
- Закрепы, альбомы скринов, описание пилота — без смешения с личной лентой.
- Партнёры-ядро (с ~01.07) смогут ссылаться на **группу**, а не на личную страницу.

**Схема публикации**

```
Личная страница          Сообщество «Telotron · для тренеров»
─────────────────        ───────────────────────────────────
Пост «от первого лица»   Тот же пост + скрины + закреп FAQ
«запускаю пилот»         Описание, ссылка reg, альбомы
Личка тёплым             Комментарии и входящие в группе
```

3. **Локальные группы** — Краснодар, Ейск, край (осторожно, без массовых копипаст).
4. **Церковные / семейные сети** — через **личное** представление; пост в личной ленте, не холодная рассылка.

**Запреты:** массовые рассылки незнакомым; однотипные комментарии подряд; обещания заработка до **01.08**.
