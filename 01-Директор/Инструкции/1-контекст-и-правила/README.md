# Контекст и правила

- **Сырьё и исследования** — всё, что накапливается в процессе (интервью, рынок, черновики); внутри папки порядок любой.
- **Решения** — этот каталог: слои от намерения до эксплуатации и принципов работы.
- **docs** — [`../../docs/`](../../docs/) — симлинк на **`_telotron.ru/docs/`** в Git приложения; одно дерево файлов, не копия.

Поток «решение → код»: при готовности переносите формулировки в `docs/`, здесь оставляйте ссылку при необходимости.
