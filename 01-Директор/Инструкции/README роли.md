# Роль: Руководитель компании

## Результат роли
- Компания движется к целям по выручке, устойчивости и масштабу.

## Основные задачи
- Утверждать приоритеты, ресурсы и ключевые решения.
- Проводить регулярный управленческий цикл.
- Снимать критичные блокеры между направлениями.

## KPI роли
- Выполнение квартальных целей.
- План-факт по выручке и расходам.
- Доля приоритетных инициатив, выполненных в срок.
