**Термин:** **продвигаем** — CTA → `telotron.ru` → `/i/{token}`.

---

## Решение

На Пилоте при **ограниченном времени** (solo, **5–6 ч/нед** на продвижение) **не распыляться** по соцсетям. Канон:

```text
① Личные связи (~30 тренеров)     — главный рычаг качества и activation
② ВКонтакте                       — единственный соцканал в фокусе
③ Telegram                        — по остатку сил; не наравне с VK
```

**MAX, Instagram, YouTube и др.** — **не** каналы продвижения на Пилоте. MAX — только **продукт** (вход, OTP, напоминания).

---

## ① Личные связи (приоритет №1)

| | |
|--|--|
| **Цель** | **~30 reg** · фокус-группа лояльных тренеров · честная ОС · ядро партнёров (**5–10**) |
| **Кто** | интервью MVP, церковные сети, личный круг |
| **Как** | 1:1, созвон 15–30 мин, своя `/i/{token}`; не холодный питч |
| **Ритм** | **5–7 касаний/нед** в активные недели (июнь) |
| **Июль** | поддержка **activated**; отбор партнёров-ядра |

**Не заменяется** постами VK: соцсети **усиливают** личку, в июне **не подменяют** её.

---

## ② ВКонтакте (приоритет №2 · основной соцканал)

| | |
|--|--|
| **Цель** | **~45–55 reg** из **~70** соцсетей к 31.07 |
| **Архитектура** | личный [профиль](https://vk.com/id224642120) + группа **«Telotron · для тренеров»** |
| **Ритм** | **2 поста/нед** (личное + группа); комментарии в группах тренеров **по делу** |
| **Личка** | всем откликнувшимся: «проведу за 15 минут» |
| **Старт** | **15.06.2026** |

**Операционка:** [04 · VK](../../02-Маркетолог/Исследования/Маркетинг-план%20на%20пилот/04-продвижение-vk.md) — чеклист, тексты, ритм.

## ③ Telegram (опционально · не в фокусе)

| | |
|--|--|
| **Роль** | Вторичный канал; **не** замена VK и **не** замена MAX |
| **Цель** | **~15–25 reg** — только если остаются силы после ① и ② |
| **Минимум** | канал + bio + пересылка постов **знакомым** из фокус-группы |
| **Не обязательно** | VPN-охота по чатам, 1 пост/нед строго, параллельный контент-план |

**Если времени мало:** **пропустить TG** на Пилоте — нормально для solo.

## Сознательно не делаем (Пилот)

| Канал / действие | Почему |
|------------------|--------|
| **Канал / лента в MAX** | MAX = auth/OTP в продукте, не маркетинг |
| **Instagram, YouTube, Дзен** | другой контент; видео — нет до 01.08; партнёры — со своих площадок позже |
| **Paid-реклама** | [стратегия](../../02-Маркетолог/Исследования/Стратегия%20роста%20—%20ручное%20продвижение%202026-2027.md): не раньше ~50–100 платников |
| **Новые соцсети «на пробу»** | пересмотр после пилота и опроса ядра (**Ц4-08**, ~01.08) |
