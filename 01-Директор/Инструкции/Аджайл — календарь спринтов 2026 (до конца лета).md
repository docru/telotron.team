## Роли (уточнение PO = маркетолог)

| Роль | Кто | В спринте |
|------|-----|-----------|
| **Product Owner** | **Маркетолог** | Тема спринта, backlog, **insight brief**; приоритет insight vs billing vs MAX |
| **Delivery** | Разработчик | Оценка capacity, delivery, техдолг |
| **Ведущий процесса** | Пока директор | Календарь, planning/retro, «задачи только из backlog» |
| **Go/no-go вех** | **Директор** | 01.06, 01.08, 01.09 — может **veto** scope, не подменяет PO в daily |
| **QA / DoD** | Тестировщик | Приёмка по scope |
| **Architecture** | Архитектор | Review billing/partner/MAX до planning 4–5 |
| **Insights** | Маркетолог + поддержка | Brief, M14, интервью (PO совмещает продукт и исследования) |

## Церемонии (шаблон на спринт)

См. [регламент](Аджайл%20—%20процесс%20Telotron%20(регламент).md) — потолок **≤ 2 ч/спринт**.

| Событие | Когда | Кто |
|---------|-------|-----|
| Insight brief | Чт–пт перед planning | PO (маркетолог) |
| Planning | Пн, день 1 | PO, dev (+ QA/архитектор по приглашению) |
| Async-статус | Каждые 1–2 раб. дня | Dev/PO → 3 строки в тикете |
| Retro | Последний раб. день | PO, dev |
| **Нет** | — | Weekly sync (пока solo); demo; daily standup |

**Отпуск 06.07–12.07:** planning **не проводим**; **on-call нет**; стабилизация prod — **до 05.07**.
