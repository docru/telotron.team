# Идеи

Наброски **без спринта** и **без T-XXX** — зафиксированы, но ещё не в [`бэклог/`](../бэклог/) как тикеты.

## Не путаем

| Папка | Что кладём |
|-------|------------|
| [`insight-brief/`](../insight-brief/) | **Раз в 2 нед.** синтез болей с **evidence** (интервью, M14) — вход для PO |
| **`идеи/`** | Любая **идея**: продукт, маркетинг, процесс — **без** обязательства и **без** T-XXX |
| [`бэклог/`](../бэклог/) | **Тикет** с критериями готовности; PO/директор согласовали «брать в работу» |

---

## Жизненный цикл

```text
  insight brief ──► идеи (idea) ──► бэклог (T-XXX) ──► в-работе ──► сделано
        │                │                  │
        │                └──── отменено ◄───┘ (директор)
        └─ сразу в бэклог, если уже ясный scope
```

| Переход | Кто | Критерий |
|---------|-----|----------|
| → **идеи** | Любой | Зафиксировать мысль; файл по [`_шаблон-идеи.md`](_шаблон-идеи.md) |
| идеи → **бэклог** | **PO** или директор | Есть **зачем**, **критерии**, связь с вехой; выдан **T-XXX** ([`_шаблон-тикета.md`](../_шаблон-тикета.md)) |
| идеи → **отменено** | Директор | Не делаем; причина в журнале идеи |
| insight → идеи / бэклог | PO | На planning или в brief |

**Правило спринта:** в scope попадает только из **`бэклог/`** (или hotfix P0). Из **`идеи/`** — **только после** «созревания» на planning/refinement.

## Когда класть в `идеи/`, а не в `бэклог/`

| `идеи/` | Сразу `бэклог/` |
|---------|------------------|
| «А что если…» без оценки | PO/директор уже сказали «делаем в спринте N» |
| Нет критериев готовности | Есть T-XXX, критерии, владелец |
| Ждёт insight / юриста / ADR | Блокер снят, scope ≤ 1 спринта |
| Несколько вариантов решения | Один выбранный вариант |
