## Как снимать (черновик)

1. Войти в Pro на **телефоне** (основной формат для VK) или desktop + кроп под мобилу.
2. Демо: 1–2 клиента, 2–3 занятия, один план — **без** реальных ФИО/телефонов посторонних; можно «Тест Клиент» / обезличить.
3. Статус-бар / лишнее — по возможности убрать; без debug-оверлеев.
4. Единый стиль: светлая тема Pro, без посторонних вкладок.
