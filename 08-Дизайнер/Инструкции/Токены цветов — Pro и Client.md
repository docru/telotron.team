# Токены цветов — Pro и Client

**Статус:** канон для MVP (макеты Figma, CSS-переменные, PWA `theme-color`).  
**Критерий:** соответствие **задачам зон**, не личному вкусу.  
**Связь:** [Решения совещания — требования UI](Решения%20совещания%20—%20требования%20UI.md) · [Стартовые выводы по бренду](Стартовые%20выводы%20по%20бренду,%20логотипу%20и%20цветам.md) (логотип, нейминг).

---

## 1. Задачи, которым служит палитра

| Зона | Задачи UI |
|------|-----------|
| **Pro** | Быстрое сканирование списков и календаря; иерархия «важно сейчас»; доверие к данным; предупреждения без паники; **визуально не Client** |
| **Client** | Один главный сценарий на экран; позитивный CTA (запись, подтверждение); **офлайн** без ощущения поломки; сопровождение, не «второй офис» |
| **Обе** | Ошибки / успех / warning читаются **одинаково**; контраст на мобиле (~430px); тариф Light — через copy и gate, **не** другой скин |

---

## 2. Принципы (обязательные)

1. **Два primary:** синий только в Pro, зелёный только в Client (интерактив и brand-акцент зоны).
2. **Одна нейтральная шкала** текста и фонов в обеих зонах (`foreground`, `muted`, `destructive`, `warning`).
3. **Client — одна hue-семья (зелёная)** для CTA; без равноправных бирюзы и розового в системных токенах.
4. **Pro — янтарь только warning**, не primary и не навигация.
5. **Офлайн (Client)** — нейтральный `offline`, не «битый зелёный».
6. **Календарь** — семантические токены слотов/типов занятий, не только `primary`.
7. **Публичная витрина** — градиент blue→green допустим; **внутри PWA** градиенты не использовать.

## 3. Общие семантические токены (Pro + Client)

| Токен | HEX | Использование |
|-------|-----|----------------|
| `foreground` | `#0F172A` | Основной текст, заголовки |
| `muted-foreground` | `#64748B` | Подписи, вторичный текст, метки |
| `border` | `#E2E8F0` | Разделители, обводка карточек |
| `destructive` | `#DC2626` | Ошибки, 409, отмена с последствиями, критичные действия |
| `destructive-foreground` | `#FFFFFF` | Текст на destructive-кнопке |
| `warning` | `#D97706` | «Требует внимания», не доставлено, истекает срок |
| `warning-subtle` | `#FFFBEB` | Фон баннера warning |
| `success` | `#16A34A` | Успех (тост, галочка); в Client совпадает с primary |
| `offline` | `#94A3B8` | Иконка/текст «нет сети», обводка некликабельных слотов |
| `offline-subtle` | `#F1F5F9` | Фон offline-баннера |

**Не использовать в UI:** `#FB7185` (розовый) как системный CTA или ошибку; `#06B6D4` (бирюза) в Client как второй brand-акцент.

## 4. `theme-pro` (Телотрон Pro)

| Токен | HEX | Задача |
|-------|-----|--------|
| `primary` | `#1D4ED8` | CTA, активная навигация, ссылки, акцент иконок |
| `primary-hover` | `#1E40AF` | Hover/focus кнопок и ссылок |
| `primary-foreground` | `#FFFFFF` | Текст на заливке primary |
| `primary-subtle` | `#EFF6FF` | Фон выделенной строки, badge, hover карточки |
| `background` | `#F8FAFC` | Shell workspace |
| `card` | `#FFFFFF` | Карточки, панели, pop-up |
| `ring` | `#93C5FD` | Focus ring (доступность) |

### PWA / manifest

- `theme_color`: `#1D4ED8`
- `background_color`: `#F8FAFC`

### Календарь (семантика, Pro)

| Токен | Значение | Задача |
|-------|----------|--------|
| `calendar-today-ring` | `primary` 30% opacity ring | Текущий день |
| `calendar-entry-individual` | `card` + `border` | Индивидуальное занятие |
| `calendar-entry-group` | `primary-subtle` + `primary` border | Групповое занятие |
| `calendar-entry-emphasis` | `primary-subtle` + medium weight text | Выбранное / сегодня с занятиями |

### Правила Pro

- Текст основного контента — **`foreground`**, не `primary`.
- **`warning`** — статусы напоминаний (M4), предупреждения; **не** кнопка «Пригласить клиента».
- Тариф Light: gate-компонент, **тот же** `theme-pro`.

## 5. `theme-client` (Телотрон Client)

| Токен | HEX | Задача |
|-------|-----|--------|
| `primary` | `#16A34A` | **Единственный** цвет CTA: «Записаться», «Подтвердить», главные кнопки |
| `primary-hover` | `#15803D` | Hover/focus primary-кнопок |
| `primary-foreground` | `#FFFFFF` | Текст на заливке primary |
| `primary-subtle` | `#F0FDF4` | Фон блока «Сегодня», мягкое выделение (не кнопка) |
| `background` | `#FFFFFF` | Shell, главная |
| `card` | `#FFFFFF` | Карточки занятий, планов |
| `ring` | `#86EFAC` | Focus ring |

### PWA / manifest

- `theme_color`: `#16A34A` (**не** синий Pro)
- `background_color`: `#FFFFFF`

### Календарь (семантика, Client)

| Токен | Значение | Задача |
|-------|----------|--------|
| `calendar-slot-free` | outline `primary` | Свободный слот, онлайн, кликабельный |
| `calendar-slot-busy` | `muted` fill | Чужое / недоступное |
| `calendar-slot-booked` | `primary-subtle` | Моя запись |
| `calendar-slot-offline` | `offline` + opacity 40% | Офлайн: **некликабельно**, не зелёный «сломанный» |

### Правила Client

- Ссылки в тексте — `primary` только если это действие; иначе `foreground` + underline.
- Ошибки и 409 — **`destructive`**, не розовый и не зелёный.
- Бирюзу `#06B6D4` — **не в MVP**; при появлении метрики «вода» — отдельная иконка метрики, не brand.
- Тариф тренера **не меняет** `theme-client`.

## 7. Реализация в коде (ориентир для разработчика)

**Канон для SPA:** CSS-переменные shadcn на корне зоны (`html` / `#app`), имена как в таблицах выше.

**Сейчас расходится с каноном** (проверить при вёрстке):

- `resources/views/pro/layout.blade.php` — `--pro-accent: #1d4ed8` ✓
- `resources/views/client/layout.blade.php` — `--client-accent: #1d4ed8` ✗ → `#16a34a`
- `PwaZoneShellService::theme_color` для Client — привести к `#16A34A`

**Workspace Vue:** классы `text-primary`, `bg-primary` должны наследовать зонную тему из layout / entry, не хардкод синего в Client.

## 8. Figma

1. Два набора переменных: **Pro** / **Client** + общий **Semantic**.
2. Стили: `Button/Primary`, `Button/Destructive`, `Banner/Offline`, `Banner/Warning`, `Calendar/Slot-*`.
3. Референс-экраны для проверки контраста: Pro — список клиентов + неделя календаря; Client — главная online + тот же offline.
