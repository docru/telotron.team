# Voice Pro и Client — канон фраз MVP

**Статус:** канон микрокопи (май 2026)  
**Реализация в коде:** `resources/ts/shared/telotron-theme.ts` → `voiceCopy`  
**Тон:** Pro — **вы**; Client — **ты**; CTA и заголовки вкладок — **без** местоимения.

---

## Pro

| Ситуация | Текст |
|----------|-------|
| Offline, календарь | Изменения расписания доступны только онлайн |
| Offline, общий | Проверьте подключение к сети |
| 409, слот | Этот слот уже занят. Выберите другое время |
| Empty, клиенты | Пока нет клиентов |
| Empty, lead | Пригласите первого — ссылка займёт минуту |
| CTA инвайт | Пригласить клиента |
| Список, счётчик | {n} клиентов |
| Список, подпись | Следующее: {дата, время} · Нет запланированных занятий |
| Копировать ссылку | Ссылка скопирована |
| Отмена занятия | Укажите причину отмены |
| Успех отмены | Занятие отменено |
| Gate Light | Групповые занятия доступны на тарифе Профи |

| Общая ошибка | Произошла ошибка. Попробуйте ещё раз. |

## Client

| Ситуация | Текст |
|----------|-------|
| Приветствие | Привет, {имя} |
| Offline, запись | Запись на занятие — при подключении к сети |
| Offline, общий | Проверь подключение к сети |
| 409, слот | Этот слот уже занят. Выбери другое время |
| Календарь, lead | Без сети можно смотреть расписание. Записаться — когда появится интернет. |
| Подсказка офлайн в карточке | Подключись к сети, чтобы записаться |
| Empty, слоты | Нет свободных слотов в этот день |
| Empty, занятия на главной | Ближайших занятий нет |
| Empty, планы | Тренер пока не назначил планов |
| M4, карточка | Подтверди занятие {когда} |
| CTA | Подтвердить · Записаться · Сохранить |
| Успех записи | Ты записан на {дата, время} |
| Синхронизация | Данные сохранятся, когда появится сеть |
| Онбординг шаг 1 | Тебя пригласил тренер {имя} |
| Анкета | Заполни анкету — откроются календарь и планы |
| Общая ошибка | Произошла ошибка. Попробуй ещё раз. |

## Не использовать в UI

- зона Pro / Client (кроме мелкой подписи Pro|Client в header)
- API, ТЗ, M2, `#123`, WebAuthn (в основном потоке — заменить на «ключ входа» / «код из Telegram или письма»)
- активных (без пояснения)
- Лента (для календаря — «Список» / «Расписание дня»)
- Конфликт (без пояснения)
