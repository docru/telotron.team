# Проверка текстов — понятность MVP

**Дата:** май 2026  
**Охват:** приоритетные экраны § [Спецификация экранов MVP](Спецификация%20экранов%20MVP.md), shell, вход/установка, смежные разделы в коде.  
**Критерии:** понятно без контекста разработчика; тон Pro «вы» / Client «ты»; без жаргона «зона», «API», «#id»; CTA глагол + объект; офлайн не как «сломалось».

**Легенда:** ✅ ок · ⚠️ улучшить · ❌ исправить обязательно

---

## 1. Сводка

| Категория | Найдено |
|-----------|---------|
| ❌ Критично | Client-экраны с «вы»; жаргон «зона Client»; технические подписи `Тренер #id` |
| ⚠️ Средне | Разные названия одного раздела; пустые/технические empty; длинные пояснения профиля |
| ✅ Хорошо | `voiceCopy` в theme; инвайты Pro (после правки); offline-баннеры; отмена с причиной |

**Канон фраз:** [Voice Pro и Client](Voice%20Pro%20и%20Client.md) (создан по итогам проверки).

## 2. Shell и навигация

| Место | Текст | Оценка | Рекомендация |
|-------|-------|--------|--------------|
| Header | Телотрон · Pro / Client | ✅ | Подпись зоны оставить (узнаваемость продукта) |
| Tab Pro | Клиенты · Календарь · Ещё | ✅ | |
| Tab Client | Главная · **Календарь** · Планы | ⚠️ | В полном меню было «Записи» — унифицировать **Календарь** |
| Страница Client tab | H1 **«Записаться»** | ⚠️ | Tab «Календарь», экран «Записаться» — путаница. H1: **«Календарь»**, подзаголовок: «Выбери слот и запишись на занятие» |
| Offline Pro | Изменения расписания только онлайн | ✅ | |
| Offline Client (запись) | Запись при подключении к сети | ✅ | Короче канона — допустимо |
| Sync bar | Нет сети / Синхронизация… / Конфликт | ⚠️ | «Конфликт» без пояснения → «Данные на сервере новее. Обнови экран» (Client: «обнови») |
| `<title>` blade | «зона Pro/Client» | ⚠️ | Для вкладки браузера: «Телотрон Pro» без «зона» |

## 3. Pro — приоритетные экраны

### 3.1. Список клиентов

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| Пригласить клиента | ✅ | |
| Клиенты / N активных | ⚠️ | «N активных» неясно → **«N клиентов»** |
| Подпись строки: телефон вместо след. занятия | ❌ | Показывается телефон; по спеке — **«Следующее: Пт, 10:00»** или «Нет занятий» |
| Empty: Пока нет клиентов + lead | ✅ | |
| Загрузка списка… | ✅ | |

### 3.2. Карточка клиента

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| К списку клиентов | ✅ | |
| Карточка клиента (подзаголовок) | ⚠️ | Лишнее — убрать или заменить на пол · возраст в шапке |
| Вкладки: Календарь · Измерения · Программы | ⚠️ | «Программы» vs ТЗ «Планы» — **Планы** или **Тренировки** единообразно |
| Клиент #N (fallback) | ⚠️ | Только для отладки; пользователю: **«Клиент без имени»** |

### 3.3. Календарь (Pro)

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| H1 Календарь + lead про сеть | ✅ | |
| Режимы: Месяц / Неделя / День / **Лента** | ⚠️ | «Лента» ≈ соцсеть → **«Список»** или **«Agenda»** → **«Расписание дня»** |
| Новое занятие | ✅ | |
| Причина отмены * | ✅ | |
| Индивидуальное (пока недоступно в API) | ❌ | Убрать «API» → **«Скоро»** или скрыть radio |
| Укажите причину… (ТЗ M2) | ⚠️ | Убрать «ТЗ M2» из UI → только пользовательский текст |
| Управление группами | ✅ | |

### 3.4. Инвайты

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| Ссылка для клиента | ✅ | |
| …привяжется к вам **в зоне Client** | ❌ | → **«…откроет приложение Телотрон Client и привяжется к вам»** |
| Партнёрская ссылка + caption | ✅ | |
| Скопировано / Скопировать | ✅ | |
| Нет активной ссылки | ⚠️ | Добавить: **«Ссылка появится после сохранения профиля»** (если так по логике) |

### 3.5. Ещё / Группы / Планы / Профиль (наследуют shell)

| Экран | Замечание |
|-------|-----------|
| Ещё | «Планы питания» — ок для диетолога; тренеру фитнеса может сбивать → позже «Планы» |
| Группы | confirm-диалоги длинные, но понятные ✅ |
| Профиль Pro | Блок про «профиль тренера не создан» — длинно ⚠️; сократить до 2 предложений |
| Профиль | «Данные аккаунта и настройки зоны» → **«…настройки аккаунта»** |

## 4. Client — приоритетные экраны

### 4.1. Главная

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| Привет, {имя} | ✅ | |
| Сегодня (M7-lite) | ⚠️ | Сейчас вода/шаги/сон — шире M7-lite в спеке; подписать блок: **«Дневник дня»** или сузить поля |
| Есть несинхронизированные данные | ⚠️ | → **«Данные сохранятся, когда появится сеть»** |
| Записаться / Измерения (ссылки) | ✅ | |
| Нет блока M4 «Подтверди занятие» | ⚠️ | Добавить при активном запросе (спека §5) |
| Нет «Мои планы» на главной | ⚠️ | В спеке — карточка планов; сейчас только tab «Планы» |

### 4.2. Календарь / запись

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| H1 Записаться + **Выберите** слот | ❌ | Client: **«Выбери»**; H1 лучше **«Календарь»** |
| В офлайне только просмотр | ⚠️ | → **«Без сети можно только смотреть расписание»** |
| Мои записи / Свободные слоты | ✅ | |
| Подпись: Тренер #N | ❌ | → **имя тренера** с API |
| 409 conflict (длинный, на «вы») | ❌ | Канон: `voiceCopy.client.slotConflict` |
| Подключитесь к сети для записи | ❌ | → **«Подключись к сети, чтобы записаться»** |
| Отмена записи недоступна (не запланировано) | ⚠️ | → **«Отменить эту запись нельзя»** |
| Записаться / Отменить запись (кнопки) | ✅ | CTA без местоимения |

### 4.3. Онбординг / вход / установка

| Текст | Оценка | Рекомендация |
|-------|--------|--------------|
| Профиль: анкета «По правилам сервиса…» | ⚠️ | Client: короче и на «ты» — **«Заполни анкету — потом откроется календарь и планы»** |
| Install: «зона Client», «веб-клиент» | ⚠️ | Пользователю: **«Телотрон Client»**, «приложение на главный экран» |
| Login Passkey/OTP | ✅ | Формальный «вы» на входе допустим (как юридический контур) |
| Invite landing | Только редирект | ✅ |

### 4.4. Планы / измерения / дневник питания

| Экран | Замечание |
|-------|-----------|
| План питания | «Тренер» fallback ✅ |
| Дневник питания | Вне приоритета MVP-главной; названия понятные ✅ |
| Измерения | «Введите число» — ок ✅ |

## 5. Общие компоненты и ошибки

| Ситуация | Pro (канон) | Client (канон) |
|----------|-------------|----------------|
| Нет сети | Проверьте подключение к сети | Проверь подключение к сети |
| 409 слот | …Выберите другое время | …Выбери другое время |
| Общая ошибка API | Произошла ошибка. Попробуйте ещё раз. | …Попробуй ещё раз — **зона-aware** (доработать `formatTelotronApiError`) |
| Gate Light | Групповые занятия доступны на тарифе Профи | — |
